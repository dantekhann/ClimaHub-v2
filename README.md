🌤️ Meu Clima Realtime — V2
Um aplicativo de previsão do tempo dinâmico e responsivo, desenvolvido em JavaScript puro (Vanilla JS). O app permite buscar informações meteorológicas de qualquer cidade do mundo em tempo real, utilizando um design Dark Neon com efeitos de Glassmorphism.

🚀 Funcionalidades
Busca Dinâmica: Traduz o nome da cidade para coordenadas geográficas automaticamente via Geocoding API.

Dados em Tempo Real: Consome a API Open-Meteo para fornecer temperatura, velocidade do vento e condições climáticas.

Tradução de Condições: Converte códigos técnicos (WMO) em descrições amigáveis e ícones (emojis), como "sem chuva por enquanto".

Interface Responsiva: Design moderno com efeitos de vidro, animações de entrada e suporte à tecla Enter para busca rápida.

Identidade Visual Ativa: Cores de destaque (Neon e Azul) que se adaptam conforme o clima da cidade pesquisada.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica e moderna.

CSS3: Estilização avançada com Glassmorphism (backdrop-filter), Gradientes e Flexbox.

JavaScript (ES6+): Lógica assíncrona (async/await) e manipulação dinâmica de DOM.

Open-Meteo API: Fonte de dados meteorológicos e geocodificação gratuita.
